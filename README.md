#  VocabVault

VocabVault is a modern vocabulary flashcard learning platform built with React, TypeScript, Tailwind CSS, and Framer Motion. It helps users transform vocabulary lists into interactive flashcards, organize words into custom decks, track learning progress, and improve retention through repeated practice.

---

##  Features

### Deck Management

* Create custom vocabulary decks
* Name and organize multiple decks
* Edit and delete decks
* Search and manage decks easily

### Intelligent Vocabulary Parsing

Supports multiple input formats:

```text
aberration - a departure from what is normal
benevolent : well meaning and kindly
```

* Handles uppercase and lowercase words
* Supports both `-` and `:` separators
* Validates input before generating flashcards
* Provides user-friendly error messages for invalid content

### Interactive Flashcards

* Flip cards to reveal meanings
* Navigate through flashcards seamlessly
* Mark words as Known or Unknown
* Restart and revisit study sessions

### Learning Progress Tracking

* Tracks mastered words
* Tracks words requiring review
* Records study performance across sessions
* Updates progress dynamically based on user responses

### Study Session Management

* Start, pause, resume, and end sessions
* Continue learning from where you left off
* Persistent session storage using Local Storage

### Modern User Experience

* Responsive design for desktop and mobile
* Dark and light theme support
* Collapsible sidebar navigation
* Smooth animations and transitions
* Clean and intuitive interface

##  Example Vocabulary Input

```text
aberration - a departure from what is normal or expected
benevolent - well meaning and kindly
candid - truthful and straightforward
diligent - showing care and effort in work
ephemeral - lasting for a very short time
```

The application automatically converts these entries into interactive flashcards.

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/snikitha05/VocabVault.git
cd VocabVault
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
